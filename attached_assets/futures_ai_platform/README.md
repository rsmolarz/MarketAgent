# Futures AI Forecast Platform (Replit)

This repo contains a modular, auditable futures forecasting system:
- IBKR front-month resolution + rollover
- Regime-aware ensembles
- Macro risk gating
- Full attribution + performance tracking
- Daily HTML email + CSV
- Weekly calibration summary + CSVs

## How to run
1. Add env vars (SMTP, IBKR, calendar APIs)
2. Run `python main.py`
